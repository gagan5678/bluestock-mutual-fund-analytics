
# Bluestock Mutual Fund Analytics

## Project Overview

The Bluestock Mutual Fund Analytics project is an end-to-end data analytics solution designed to evaluate mutual fund performance, risk exposure, portfolio diversification, and investor behavior. The project leverages Python, SQL, SQLite, and Power BI to transform raw financial data into actionable business insights.

## Business Problem

Mutual fund investors often struggle to compare funds effectively using only returns. This project provides deeper analysis using risk-adjusted performance metrics, portfolio concentration measures, and investor behavior analytics.

## Project Objectives

* Analyze mutual fund performance.
* Calculate risk-adjusted return metrics.
* Evaluate portfolio concentration.
* Study investor transaction behavior.
* Build interactive dashboards.
* Generate business recommendations.

## Key Insights

* High Sharpe Ratio funds delivered better risk-adjusted returns.
* Certain funds showed high concentration risk.
* Investor retention varied significantly across cohorts.
* SIP continuity improved long-term engagement.
